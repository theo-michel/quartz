    # Value Function

    Discount factor $\gamma \in[0,1]$ helping to make it desirable to achieve a reward ASAP $$ V_{\pi}=\mathbb{E}[G_{t}|S_{t},\pi] $$ $G_{t} = R_{t+1}+\gamma G_{t+1}$ $v_{\pi}(S)=\mathbb{E}[R_{t+1}+\gamma v_{\pi}(S_{t+1})|S_{t}=s,A_{t}\approx\pi(s)]$ with $,A_{t}\approx\pi(s)]$ means that $A_{t}$ is chosen with the policy $\pi$ This is a Bellman Equation Also exists if we want to find the optimal $v_{*}(S)= \underset{a}max{}\mathbb{E}[R_{t+1}+\gamma v_{\pi}(S_{t+1})|S_{t}=s,A_{t}=\pi(s)]$ We will almost always approximate the value functions
